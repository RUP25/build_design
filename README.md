# Build Design Projects

Premium turnkey execution website for Build Design Projects — inspired by luxury real estate design patterns.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- Framer Motion
